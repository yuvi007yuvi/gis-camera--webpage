# GIS SOS Emergency Dashboard
### Sultanganj Municipal Council | GIS-Based Emergency Support

A high-priority, interactive emergency response dashboard for real-time GIS-based SOS monitoring and life-safety management.

---

## 🚀 Key Features
1.  **GIS Response Map**: High-fidelity Leaflet.js-based mapping tracking all SOS nodes.
2.  **Emergency SOS Alert**: Full-screen emergency overlays with audible alarms and visual beaconing.
3.  **Real-Time Data**: Powered by **Firebase Firestore** for instant cloud synchronization.
4.  **Hardware Focused**: Designed specifically to ingest signals from hardware switches and dedicated SOS poles.

---

## 🛠 Project Components
-   **Dashboard** (`index.html`): The main receiver interface.
-   **Simulator** (`test-sender.html`): A field emulator to test SOS signals.

## ⚙️ Technical Requirements
-   **Backend**: Google Firebase (Firestore).
-   **Interface**: HTML5/CSS3/JS (Glassmorphic Design).
-   **Mapping**: Leaflet.js Engine.
